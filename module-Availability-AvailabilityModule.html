    • Class: AvailabilityModule

    Availability~AvailabilityModule

    The Availability module.

    Constructor

    new AvailabilityModule(app)

    Parameters:
    Name Type Description
    app ClickToDialApp

    The application object.

    Methods

    _load()

    Do an API request to get an update of the available userdestination options when the module is loaded in the background.

    _restore()

    This is called when the popup refreshes and the background already has processed all availability data.

    availabilityOptions(userdestination, selectedFixeddestinationId, selectedPhoneaccountId) → {Array}

    Build an array of availability data that can be used to build the availability select.

    Parameters:
    Name Type Description
    userdestination Object

    Contains available phoneaccounts and fixeddestinations to build the availability list from.
    selectedFixeddestinationId String

    Fixeddestination to select.
    selectedPhoneaccountId String

    Phoneaccount to select.
    Returns:
    Array -
    • Data that is used to build the select element with.

    toggleAvailabilitySelect()

    Enable or disable the availability select based on the Are you available radio button value.
