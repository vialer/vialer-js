    • Class: ModuleCalls

    AppBackground.modules.ModuleCalls

    Main entrypoint for Calls.

    Constructor

    new ModuleCalls(app)

    Parameters:
    Name Type Description
    app AppBackground

    The background application.

    Methods

    __setTransferState(sourceCallopt, active)

    Set the transfer state of a source call and update the transfer state of other calls. This method doesn't change the intended transfer status when no source Call is passed along. It just update outdated call state in that case.

    Parameters:
    Name Type Attributes Description
    sourceCall Call <optional>

    The call to update the calls status for.
    active Boolean

    The transfer status to switch or update to.

    __uaOptions() → {Object}

    Setup the initial UA options. This depends for instance on whether the application will be using the softphone to connect to the backend with or the vendor portal user.

    Returns:
    Object -

    UA options that are passed to Sip.js

    _emptyCall(opts) → {Call}

    Check if there is a new call ready to be used. Requires some bookkeeping because of the settings that can change in the meanwhile.

    Parameters:
    Name Type Description
    opts Object

    Options to pass.

    Properties
    Name Type Attributes Description
    type String <optional>

    The type of call to find.
    number String <optional>

    The number to call to.
    Returns:
    Call -
    • A new or existing Call with status new.

    _formatSdp(sessionDescription) → {SessionDescription}

    Reformat the SDP of the SessionDescription when setting up a connection, so we can force opus or G722 codec to be the preference of the backend.

    Parameters:
    Name Type Description
    sessionDescription SessionDescription

    A Sip.js SessionDescription handler.
    Returns:
    SessionDescription -

    A SessionDescription with a modfied sdp object.

    _initialState() → {Object}

    Initializes the module's store.

    Returns:
    Object -

    The module's store properties.

    _restoreState(moduleStore)

    Restore stored dumped state from localStorage.

    Parameters:
    Name Type Description
    moduleStore Object

    Root property for this module.

    _userAgent() → {String}

    Build the useragent string to identify Vialer-js with. The format is Vialer-js/<VERSION> (<OS/<ENV>) <Vendor>

    Returns:
    String -
    • Useragent string.

    activateCall(callopt, holdOthersopt, unholdOwnopt) → {Call|Boolean}

    Set the active state on the target call, un-hold the call and put all other calls on-hold.

    Parameters:
    Name Type Attributes Default Description
    call Call <optional>

    A Call to activate.
    holdOthers Boolean <optional>
    		 true

    Unhold the call on activation.
    unholdOwn Boolean <optional>
    		 false

    Unhold the call on activation.
    Returns:
    Call | Boolean -
    • The Call or false.

    activeCall(ongoing) → {Call|null}

    Parameters:
    Name Type Default Description
    ongoing Boolean false

    Whether to check if the call is ongoing or not.
    Returns:
    Call | null -
    • the current active ongoing call or null.

    callAction(action)

    A loosely coupled Call action handler. Operates on all current Calls. Supported actions are: accept-new: Accepts an incoming call or switch to the new call dialog. deline-hangup: Declines an incoming call or an active call. hold-active: Toggle hold on the active call.

    Parameters:
    Name Type Description
    action String

    The action; accept-new or decline.

    connect()

    Initialize the SIPJS UserAgent and register its events.

    deleteCall(call)

    Take care of cleaning up an ending call.

    Parameters:
    Name Type Description
    call Call

    The call object to remove.

    disconnect(reconnect)

    Graceful stop, do not reconnect automatically.

    Parameters:
    Name Type Default Description
    reconnect Boolean true

    Whether try to reconnect.

    toString() → {String}

    Generate a representational name for this module. Used for logging.

    Returns:
    String -
    • An identifier for this module.
